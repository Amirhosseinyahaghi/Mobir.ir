## معرفی پروژه

**Mobir.ir** یک پلتفرم دو‌طرفه برای خرید و فروش گوشی‌های همراه است که کاربران را به هم متصل می‌کند تا بتوانند محصولات خود را به فروش برسانند و محصولات دیگران را بخرند.

### اهداف پروژه
✅ ایجاد یک پلتفرم امن و قابل اعتماد برای تجارت موبایل  
✅ ارائه رابط کاربری سهل‌الاستفاده و ریسپانسیو  
✅ فراهم آوردن سیستم مدیریت کامل برای مدیران  
✅ پشتیبانی کامل از فارسی و RTL

## نصب و راه‌اندازی

### پیش‌نیازها
```
- سرور وب با پشتیبانی PHP 7+
- MySQL Server 5.7+
- کتاب‌خانه‌های PHP مانند MySQLi
```

### مراحل نصب

#### 1️⃣ استخراج فایل‌ها
```bash
git clone https://github.com/Amirhosseinyahaghi/Mobir.ir.git
cd Mobir.ir
```

#### 2️⃣ ایجاد پایگاه داده
```sql
-- فایل SQL موجود است: mobir (1).sql
mysql -u root -p < "mobir (1).sql"
```

#### 3️⃣ تنظیم فایل پیکربندی
پایگاه داده را در فایل `config.php` تنظیم کنید:

```php
<?php
$conn = mysqli_connect("localhost", "root", "root", "mobir");
mysqli_query($conn, "SET NAMES 'utf8'");
mysqli_query($conn, "SET CHARACTER SET 'utf8'");
mysqli_query($conn, "SET character_set_connection = 'utf8'");
?>
```

**نکته**: رمز عبور را با رمز عبور MySQL خود جایگزین کنید.

#### 4️⃣ آپلود بر روی سرور
تمام فایل‌ها را به دایرکتوری پروژه (مثل `/var/www/html/mobir/`) آپلود کنید.

#### 5️⃣ دسترسی به پروژه
```
http://localhost/mobir/index.php
```

## آموزش استفاده

### برای کاربران عام

#### ✅ ثبت‌نام
1. روی دکمه **"ثبت نام"** در بالا راست کلیک کنید
2. فرم را پر کنید (نام، ایمیل، رمز عبور)
3. روی دکمه **"ثبت"** کلیک کنید

#### 🔍 جستجو برای محصول
1. از جعبه جستجو در صفحه اصلی استفاده کنید
2. نام یا مدل محصول را وارد کنید
3. روی دکمه **"برو"** کلیک کنید

#### 🛒 خرید محصول
1. محصول مورد نظر را انتخاب کنید
2. روی دکمه **"خرید"** کلیک کنید
3. اطلاعات حساب خود را تأیید کنید

#### 📤 فروش محصول
1. پس از ورود، به **"مدیریت"** رفتید
2. روی **"ثبت محصول"** کلیک کنید
3. مشخصات محصول را وارد کنید:
   - نام محصول
   - مدل
   - قیمت
   - عکس محصول
4. روی **"ثبت"** کلیک کنید

## رفع مشکلات رایج

### ❌ مشکل: "Connection failed"
**حل:** اطمینان حاصل کنید که:
- MySQL سرور در حال اجرا است
- نام کاربری و رمز عبور صحیح است
- نام پایگاه داده صحیح است

### ❌ مشکل: صفحه 404 نمایش می‌دهد
**حل:** 
- فایل‌ها را به درستی آپلود کنید
- نام دایرکتوری را بررسی کنید

### ❌ مشکل: متن‌های فارسی نمایش نمی‌دهند
**حل:**
- Charset را در header قرار دهید
- فایل config.php را بررسی کنید
